sublime text在当前文件所在目录运行cmd

2019年04月16日 224 字 大概 1 分钟 阅读量： loading...
  1. 1. 安装插件Cmd Caller
  2. 2. 配置
  3. 3. 添加快捷键
  4. 4. 运行演示

安装插件Cmd Caller

  1. ctrl + shift + p，在跳出窗口搜索Package Control: Install Package

3.2.点击Package Control: Install Package，等待搜素框跳出后搜素插件cmd caller，并安装。

配置

依次点击：preferences->Package Settings->Cmd Caller->Settings

在cmd_caller.sublime-settings —— User下添加代码：

{
    "windows": {
        "default": "CMD",
        "apps":{
            "CMD": {
            "name": "Run cmd.exe Here",
            "cmd": "cmd.exe /s /k pushd \"${file_path}\""
          }
      }
    }
}

添加快捷键

我这里设置的是ctrl+shift+c，你可以按你的需要更改。
依次点击：Preferences->Key Blindings

在Default.sublime-keymap —— User文件中添加代码(数组里面添加):

{ "keys": ["ctrl+shift+c"], "command": "cmd_caller_default" }
