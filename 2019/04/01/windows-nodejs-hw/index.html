Windows下安装node.js并创建第一个程序

安装node.js

node.js下载地址：https://nodejs.org/en/download/

进入链接，点击箭头处下载安装包：

点开安装包，Next：

等待安装，最后Finish


它帮我们已经配置好了环境变量，进入cmd中检验，输入node --version：

第一个程序（helloworld.js）

var http = require('http');
http.createServer(function(request, response){
  response.writeHead(200, {'Content-Type':'text/plain'});
  response.end("Hello World");
}).listen(8887);
console.log('Server running at http://127.0.0.1:8887/');

进入cmd，尝试用node来运行：
切换路径为文件所在路径，再用node运行。如果你不清楚自己的文件在哪，可以右键文件在属性中复制。
我这里是将文件helloworld.js放在了桌面：

cd desktop(此处应为你自己的文件路径)
node helloworld.js（此处应为你保存的文件名字）

打印信息出现，运行成功。
最后复制链接http://127.0.0.1:8887/在浏览器访问：
