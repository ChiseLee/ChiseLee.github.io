个人感受

这是第三次观看memory snow了，今年一月re的新编集更新，这周更完的内容正好接着剧场版，便在b站重温了一遍。

Re：从零开始的异世界生活对我来说是一部比较特别的作品，如果让我评价它的动画，我会说治愈也致郁，

而如果是评价ova memory sonw，我只会说：治愈。

作为动画的粉丝，我看得十分舒服，那种安宁美好的氛围，正是许多“死宅”所追求的环境。

看过动画的人也应该知道，这个“约会之晨”的到来，是昴经过了无数痛苦的回忆才换来的。

（出自：Re：从零开始的异世界生活动画第七话：莱月.昴重新开始）

如果你想看这部ova却不了解动画内容，建议在看完第一季11话雷姆或是新编集第6话鬼上身的作法/雷姆后再观看，以获得较好的观看体验。
